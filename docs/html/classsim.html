Initialize the simulation.

Allocates runtime memory for vehicle state, charger state, and copies vehicle type data into the simulation.

Also initializes random number generators and calculates per-step runtime values.

Parameters
typesArray of available vehicle type definitions.
num_typeNumber of vehicle types in the types array.
statsPointer to statistics output array.
num_statNumber of vehicle instances to simulate.
num_chargerNumber of available charging stations.
