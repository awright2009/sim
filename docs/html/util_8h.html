Detailed Description

Utility functions and simulation configuration constants.

This header defines:

  • Default simulation configuration limits
  • CSV formatting constants
  • Utility functions for loading vehicle data and exporting statistics

int read_vehicle_data ( vehicle_data_t * types,
char * filename )

Read vehicle configuration data from a CSV file.

Opens the specified CSV file, reads all vehicle data entries, and populates an array of vehicle_data_t structures with the parsed values. Each row in the CSV corresponds to one vehicle definition.

Expected CSV column order:

  • Column A: Company name
  • Column B: Cruise speed
  • Column C: Battery capacity
  • Column D: Charge time
  • Column E: Cruise energy usage
  • Column F: Passenger count
  • Column G: Fault probability
Parameters
typesPointer to an array of vehicle_data_t structures to populate.
filenamePath to the CSV file containing vehicle data.
Returns
0 on success, -1 on failure.
