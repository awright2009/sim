\doxysubsection{Detailed Description}
Fixed-\/size byte queue implementation. 

This header defines a simple FIFO queue interface using a circular buffer. The queue supports\+:
\begin{DoxyItemize}
\item Appending data to the tail
\item Inserting data at the front
\item Removing data from the head
\item Peeking at queued data without removal
\item Debug printing of queue contents 
\end{DoxyItemize}
