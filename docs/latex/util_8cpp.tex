\doxysubsection{Detailed Description}
Utility functions for simulation configuration and statistics output. 

This source file implements helper functions for\+:
\begin{DoxyItemize}
\item Reading vehicle configuration data from CSV files
\item Parsing vehicle operational parameters
\item Generating simulation statistics reports
\item Exporting per-\/vehicle and per-\/company aggregated metrics
\end{DoxyItemize}

Generated statistics include\+:
\begin{DoxyItemize}
\item Average flight time
\item Average flight distance
\item Average charging duration
\item Fault totals
\item Passenger miles traveled
\end{DoxyItemize}

CSV output files\+:
\begin{DoxyItemize}
\item vehicle\+\_\+stats.\+csv
\item type\+\_\+stats.\+csv 
\end{DoxyItemize}

Read vehicle configuration data from a CSV file. 

Opens the specified CSV file, reads all vehicle data entries, and populates an array of \doxylink{structvehicle__data__t}{vehicle\+\_\+data\+\_\+t} structures with the parsed values. Each row in the CSV corresponds to one vehicle definition.

Expected CSV column order\+:
\begin{DoxyItemize}
\item Column A\+: Company name
\item Column B\+: Cruise speed
\item Column C\+: Battery capacity
\item Column D\+: Charge time
\item Column E\+: Cruise energy usage
\item Column F\+: Passenger count
\item Column G\+: Fault probability
\end{DoxyItemize}
